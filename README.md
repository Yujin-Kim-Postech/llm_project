# insurance-rq-tree (starter)

## What is included
- `data/papers.jsonl`: 5 sample paper objects (one JSON per line)
- `labels/paper_labels.jsonl`: confirmed L1/L2/tags for the 5 papers
- `labels/review_queue.jsonl`: autogenerated L1 suggestions (you can manually fill `final_l1/final_l2`)
- `ontology.yaml`: L1/L2 standard
- `src/`: scripts

## Quickstart
```bash
python -m pip install pyyaml
python src/review_queue.py
python src/build_tree.py
```

Outputs:
- `labels/review_queue.jsonl`
- `tree.json` (D3-friendly)
